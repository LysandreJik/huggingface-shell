> HuggingFace theme for [Oh My Fish][omf-link].

## Install

```fish
$ omf theme huggingface
```

## Features

* A stargazers vizualizer
* A rad HuggingFace icon
* Previous command status.

<p align="center">

## Configuration

Only display the folder name:
```
  set -g theme_short_path yes
```
